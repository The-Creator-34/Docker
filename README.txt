# Simple Python Calculator
This is a basic Python calculator application that can perform addition, subtraction, multiplication, and division.

## How to Run the Application
1. Install Docker on your computer.
2. Pull the Docker image from Docker Hub:
   docker pull thecreator34/calculator-app:latest
3. Run the container:
   docker run -it thecreator34/calculator-app:latest
